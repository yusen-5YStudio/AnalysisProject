# eBay 电商数据分析项目(eBay 爬虫数据抓取公共数据集) Python+PowerBI

### 项目简介

```
本项目基于eBay电商商品数据，通过Python进行数据清洗与分析，并结合PowerBI构建可视化Dashboard，用于分析商品销量、品牌表现、价格结构及市场竞争情况
```

### 分析目标

```
1.分析平台整体销售情况
2.识别热销商品
3.评估品牌市场竞争力
4.探索价格与销量之间的关系
5.构建交互式数据看板
```

### 数据说明

```
数据字段包括:
	item_name:商品名称
	brand:品牌
	avg_price:平均价格
	sold_quantity:销量
	view_quantity:浏览量
	comments:商品状态(全新/二手/99新)
```

### 数据清洗

```
使用python进行数据预处理:
	1.处理缺失值
	2.统一品牌字段格式
	3.处理价格区间数据
	4.转换数据类型
	5.去除异常值
```

### 核心指标

```
总商品数量:约1118
总销售:约85k
平均价格:约229$
头部品牌:三星/苹果等
市场份额(结构):头部品牌集中明显
```

### 可视化分析

```
品牌分析(品牌销量Top10、品牌商品前10情况、品牌平均价格情况、品牌前10市场份额)
价格区间分析(价格区间销售分布、不同价格段表现)
价格 vs 销量散点图
发货国家分析
销售分析(Top10 热销商品)
```

### Power BI Dashboard 包含

```
KPI 指标卡片(品牌/价格区间/商品状态)
总商品数量、总销量、平均价格
品牌销量Top 10
品牌市场份额图
价格区间销量分析
Top10热销商品
价格 vs 销量散点图
```

### 关键业务结构

```
1.市场呈现明显头部集中(三星 & 苹果 占比高)
2.中高价商品贡献主要销量
3.商品销售呈长尾分布
4.价格对销量有明显影响
```
