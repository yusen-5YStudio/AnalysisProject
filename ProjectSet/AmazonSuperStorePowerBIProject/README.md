# SuperStore 销售数据分析仪表板

## 项目简介

```
本项目基于亚马逊的Superstore 销售数据集，使用Power BI构建销售分析仪表板，通过数据可视化展示销售表现、客户分析、产品销售情况及时间趋势，为业务分析和决策提供支持
```

## 项目目标

```
1.统计企业整体销售情况
2.分析不同区域销售表现
3.分析不同客户群体销售贡献
4.识别高销售产品及产品类别
5.观察销售额随时间变化趋势
6.实现交互式数据筛选分析
```

## 数据字段

```
本项目主要使用了以下字段:
	Order ID(订单编号)
	Order Date(订单日期)
	Customer ID(客户编号)
	Customer Name(客户名称)
	Region(区域)
	Segment(客户群体)
	Category(产品类别)
	Sub-Category(产品子类别)
	Product Name(产品名称)
	Sales(销售额)
```

## 核心指标

```
使用DAX创建一下指标:
	总销售额(Total Sales)
	总订单数(Total Orders)
	总客户数(Total Customers)
```

## 仪表板内容

| 类目           | 描述                                                         |
| :------------- | :----------------------------------------------------------- |
| 销售概览       | 展示企业整体销售情况，包括: 总销售额、总订单数、总客户数     |
| 区域销售分析   | 分析不同区域的销售贡献:West、East、Central、South。用于识别核心销售区域 |
| 客户群体分析   | 分析不同客户群体销售额贡献：Consumer、Corporate、Home Office。用于评估客户结构 |
| 产品类别分析   | 分析不同产品类别销售表现：Technology、Furniture、Office Supplies。用于识别主要销售来源 |
| 产品子类别分析 | 分析各子类别销售情况、识别高销售产品方向                     |
| 销售趋势分析   | 基于订单日期分析销售额变化趋势，观察业务发展情况             |
| Top10产品分析  | 根据销售额筛选销售额最高的产品，识别核心销售商品             |

## 技术栈

```
Power BI、DAX、数据可视化
```

## 项目成果

```
1.构建交互式销售分析仪表板
2.实现区域、客户群体及产品维度分析
3.实现销售趋势分析
4.支持切片器动态筛选
5.提示业务数据分析效率
```
