# 长沙头部电商公司招聘市场分析

### 项目简介

```
本项目基于智联、猎聘招聘平台(BOSS有反扒，影刀爬不到)，选取长沙地区头部电商互联网企业招聘信息，利用 Python 进行数据清洗与预处理，结合 Power BI 构建可视化分析看板，对企业招聘需求、薪资水平、学历要求及区域分布进行分析。通过本项目探索长沙头部企业人才需求现状，为求职者职业规划及企业招聘决策提供数据支持。
```

### 项目目标

```
- 分析长沙头部电商企业招聘需求情况
- 研究不同学历要求对应的薪资水平
- 分析各区域岗位分布特征
- 统计薪资区间分布情况
- 识别高薪岗位及人才需求方向
```

### 数据来源

```
智联、猎聘
主要字段包括：
	- 职位名称
	- 公司名称
	- 工作地点
	- 学历要求
	- 薪资范围
	- 平均薪资
```

### 数据处理流程

##### Python数据清洗

```
	- 处理缺失值
	- 删除重复记录
	- 统一学历字段格式
	- 提取薪资上下限
	- 计算平均薪资
	- 数据类型转换
```

##### Power BI可视化分析

```
构建招聘市场分析 Dashboard：
	- 学历需求分布
	- 各区域岗位需求分布
	- 各区域平均薪资分析
	- 学历与薪资关系分析
	- 薪资区间分布分析(构建了薪资区间字段)
```

### 核心分析结论

##### 学历要求分析

```
- 本科学历为主要招聘门槛
- 高薪岗位对学历要求相对更高
```

##### 薪资分析

```
- 招聘岗位主要集中在10K-20K薪资区间
- 20K以上岗位主要集中于技术及管理岗位
```

##### 区域分析

```
- 长沙高新区、岳麓区招聘需求较为集中
- 不同区域之间存在明显薪资差异
```

##### 学历与薪资关系分析

```
- 学历要求与岗位薪资呈正相关关系
- 硕士岗位平均薪资最高
```
